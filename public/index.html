🧩 LexiTrack

A local-network word game for 2–8 players

📺
Dedicated Host Screen
Open this on a laptop or TV. Shows the board, timer, and all players' word counts. Players join separately by scanning the QR code.
📱
Host + Play (Phone)
No dedicated screen? Open this on the phone running the server. You'll play and host at the same time, and a QR code lets others join.
🎮
Join as Player
Already have a host screen set up? Open this (or scan the QR code) to join as a player.

Run npm start in the lexitrack folder, then share the host URL with your TV or phone.
